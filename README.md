# Serverless Discord ChatGPT bot

## Prerequisites

- Serverless `npm i -g serverless`
- Discord bot
- OpenAI API Key
- AWS Profile setup in `~/.aws/credentials`


## Setup

1. clone
2. `mv .env.exmaple .env`
3. fill in credentials
4. `serverless deploy`

## Running locally
- `serverless offline start`


## Remove
- `serverless destroy`
